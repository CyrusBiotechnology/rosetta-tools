#This is a helper file for the include_what_you_use.py script
#
# This file lists alternative locations from which one can consider to get
# a particular header. That is if you have any of the headers B, C, D, or E, 
# you shouldn't need to include the file A. -- Or in other words, that the B/C/D/E addition shadows the A include.
#
# In general, only add a header as a primary entry here if people shouldn't
# be including the header directly. (It's better to be explicit.)
#
# Note that the A include won't be substituted if it's already present (not being slated to be added)
#
# Normally if the header which would shadow is being deleted, the shadowing won't happen,
# But if you add the comment #ADDDEL to the line, the the shadowing will stop the deletion and addition.
